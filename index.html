Controller Mod · Build Sheet

แบบฟอร์มสั่งคัสตอมคอนโทรลเลอร์

เลือกบอร์ด PCB ปุ่มหลัง ทริกเกอร์ และอนาล็อกสติ๊กตามที่ต้องการ ระบบจะคำนวณราคารวมให้อัตโนมัติที่แผงสรุปด้านข้าง (หรือด้านล่างในมือถือ)

*ต้องเลือก  ·  ข้อที่ไม่มีเครื่องหมาย * เลือกเพิ่มได้ตามต้องการ

01 Component 01 · Main Board บอร์ด PCB*

⚠ กรุณาเลือกบอร์ด PCB

02 Component 02 · Rear Paddles ปุ่มหลัง*

※ฝาหลังของคอนโทรลเลอร์ที่ส่งมาจะถูกเปลี่ยนตามจำนวนปุ่มที่เลือก โดยไม่ขึ้นกับสภาพเดิมก่อนส่ง

⚠ กรุณาเลือกจำนวนปุ่มหลัง

03 Component 03 · Trigger Module (เลือกเพิ่มได้) Clicky Trigger / Clicky Bumper
04 Component 04 · Face Buttons (เลือกเพิ่มได้) D-pad Clicky Button / Action Clicky Button

※ตัวเลือกนี้จะปรากฏเมื่อติ๊กเลือก Clicky Trigger หรือ Clicky Bumper ด้านบน

05 Component 05 · Left Stick Analog ซ้าย*

⚠ กรุณาเลือก Analog ซ้าย

06 Component 06 · Right Stick Analog ขวา*

⚠ กรุณาเลือก Analog ขวา

07 Component 07 · PS4 Shell กรอบคอนโทรลเลอร์ PS4*

※หากเลือก "ส่งจอยมาให้ทำ" กรุณาแนบคอนโทรลเลอร์ PS4 มาพร้อมพัสดุที่จัดส่ง

⚠ กรุณาเลือกตัวเลือกกรอบคอนโทรลเลอร์ PS4
